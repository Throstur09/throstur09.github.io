Uppbygging spurningar

Opnaðu þessa HTML skrá í textaritli (t.d. Notepad, VS Code).

Finndu hlutann sem byrjar á const spurningar = { í JavaScript-kóðanum.

Þar eru þrír listar: venjulegur, blessun og bolvun.

Hvert stak hefur þessa uppbyggingu:

{ spurning: "Spurningin hér?", svar: "Svarið hér", bonus: "Hvað gerist" }

Fyrir venjulegar og blessunarspurningar skaltu nota bonus (gefið við rétt svar).
Fyrir bölvunarspurningar skaltu nota refsing (gefið við rangt svar).
